R Dataset / Package wooldridge / wagepan

On this R-data statistics page, you will find information about the wagepan data set which pertains to wagepan. The wagepan data set is found in the wooldridge R package. You can load the wagepan data set in R by issuing the following command at the console data("wagepan"). This will load the data into a variable called wagepan. If R says the wagepan data set is not found, you can try installing the package by issuing this command install.packages("wooldridge") and then attempt to reload the data with the library() command. If you need to download R, you can go to the R project website. You can download a CSV (comma separated values) version of the wagepan R data set. The size of this file is about 497,233 bytes.

wagepan

Description

Data loads lazily. Type data(wagepan) into the console.

Usage

data(wagepan)
