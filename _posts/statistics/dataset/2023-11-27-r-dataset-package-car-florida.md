<div id="dataset-info">
<p>On this R-data statistics page, you will find information about the <span class="mono">Florida</span> data set which pertains to Florida County Voting. The <span class="mono">Florida</span> data set is found in the <span class="mono">car</span> R package. You can load the <span class="mono">Florida</span> data set in R by issuing the following command at the console <span class="mono">data("Florida")</span>. This will load the data into a variable called <span class="mono">Florida</span>. If R says the <span class="mono">Florida</span> data set is not found, you can try installing the package by issuing this command <span class="mono">install.packages("car")</span> and then attempt to reload the data with the <span class="mono">library()</span> command. If you need to download R, you can go to the <a href="https://www.r-project.org">R project website</a>. You can download a CSV (comma separated values) version of the <span class="mono"><a href="../assets/data/csv/dataset-62310.csv">Florida R data set</a></span>. The size of this file is about 3,375 bytes.</p><h2>Florida County Voting</h2>
<h3>Description</h3>
<p>The <code>Florida</code> data frame has 67 rows and 11 columns. Vote by county in Florida for President in the 2000 election.</p>
<h3>Usage</h3>
<pre>
Florida
</pre>
<h3>Format</h3>
<p>This data frame contains the following columns:</p>
<dl>
<dt>GORE</dt>
<dd>
<p>Number of votes for Gore</p>
</dd>
<dt>BUSH</dt>
<dd>
<p>Number of votes for Bush.</p>
</dd>
<dt>BUCHANAN</dt>
<dd>
<p>Number of votes for Buchanan.</p>
</dd>
<dt>NADER</dt>
<dd>
<p>Number of votes for Nader.</p>
</dd>
<dt>BROWNE</dt>
<dd>
<p>Number of votes for Browne (whoever that is).</p>
</dd>
<dt>HAGELIN</dt>
<dd>
<p>Number of votes for Hagelin (whoever that is).</p>
</dd>
<dt>HARRIS</dt>
<dd>
<p>Number of votes for Harris (whoever that is).</p>
</dd>
<dt>MCREYNOLDS</dt>
<dd>
<p>Number of votes for McReynolds (whoever that is).</p>
</dd>
<dt>MOOREHEAD</dt>
<dd>
<p>Number of votes for Moorehead (whoever that is).</p>
</dd>
<dt>PHILLIPS</dt>
<dd>
<p>Number of votes for Phillips (whoever that is).</p>
</dd>
<dt>Total</dt>
<dd>
<p>Total number of votes.</p>
</dd>
</dl>
