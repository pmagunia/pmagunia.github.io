<p>On this R-data statistics page, you will find information about the <span class="mono">Nerlove</span> data set which pertains to Cost Function for Electricity Producers, 1955 . The <span class="mono">Nerlove</span> data set is found in the <span class="mono">Ecdat</span> R package. You can load the <span class="mono">Nerlove</span> data set in R by issuing the following command at the console <span class="mono">data("Nerlove")</span>. This will load the data into a variable called <span class="mono">Nerlove</span>. If R says the <span class="mono">Nerlove</span> data set is not found, you can try installing the package by issuing this command <span class="mono">install.packages("Ecdat")</span> and then attempt to reload the data with the <span class="mono">library()</span> command. If you need to download R, you can go to the <a href="https://www.r-project.org">R project website</a>. You can download a CSV (comma separated values) version of the <span class="mono"><a href="../assets/data/csv/dataset-69871.csv">Nerlove R data set</a></span>. The size of this file is about 7,218 bytes.</p><h2>Cost Function for Electricity Producers, 1955</h2>
<h3>Description</h3>
<p>a cross-section from 1955 to 1955</p>
<p><em>number of observations</em> : 159</p>
<p><em>observation</em> : production units</p>
<p><em>country</em> : United States</p>
<h3>Usage</h3>
