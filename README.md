# pmagunia.github.io

This repository contains the code to generate the pages at pmagunia.com.

# Fork of Prologue theme for Jekyll

Original code can be found at:

https://github.com/chrisbobbe/jekyll-theme-prologue

Once you have Jekyll installed with the dependencies you can run:

`bundle exec jekyll build`

The HTML files will be put in the `docs` directory as per `_config.yml`.
